# Image & Video Processing Lab (Spring 2026)

This repository contains a collection of MATLAB practice sessions and a final project developed as part of the **Image and Video Processing** course during the **Spring 2026 semester**.

The work focuses on fundamental and applied concepts in image processing using:
- MATLAB
- Image Processing Toolbox
- Computer Vision Toolbox

---

## Repository Structure

The repository is organized into **7 session folders** and **1 final project**:

### Sessions

Each session folder (`Session 1`, `Session 2`, ..., `Session 7`) contains a MATLAB script implementing the concepts covered during that session.

#### **Session 1 – Introduction to MATLAB**
- Basic MATLAB syntax and operations
- Introduction to matrix and image handling

#### **Session 2 – Image Processing Toolbox Basics**
- Low-Pass Filtering (LPF)
- High-Pass Filtering (HPF)
- Discrete Fourier Transform (DFT) on images

#### **Session 3 – Frequency Domain Processing**
- Convolution in the frequency domain
- Image comparison metrics:
  - Mean Squared Error (MSE)
  - Mean Absolute Error (MAE)

#### **Session 4 – Image Enhancement Techniques**
- Histogram Equalization
- Log Transformation
- Gamma Transformation
- Image Negative (Inverse)
- Filtering techniques:
  - Gaussian Filter
  - Laplacian Filter
  - Mean Filter
  - Median Filter

#### **Session 5 – Image Compression (Lossless)**
- Huffman Encoding applied to images

#### **Session 6 – Feature-Based Object Detection**
- Object detection using the **SURF (Speeded-Up Robust Features)** method

#### **Session 7 – Image Compression (Lossy)**
- Uniform Quantization for image compression

---

### Final Project – Car Plate Detection

The repository also includes a final project focused on:

- **Automatic car plate detection**
- Image preprocessing and enhancement
- Feature extraction and detection techniques

The project demonstrates a practical application of the concepts learned throughout the course using MATLAB and relevant toolboxes.
